# myrabbitmq
RabbitMQ实战练习


### 项目概述
#### 仅仅是个demo，那也是实战级别的demo。我相信在项目里也不过如此，只是多了一些业务处理而已。
#### 消息机制：
- Fanout
- Direct
- Topic
#### 其他：
配置了消息确认机制和消息发送是否成功回调机制
### 项目结构
##### rabbitmq-common：公共模块
主要包含了几个工具类，以及消息的定义类


##### rabbitmq-consumer：RabbitMQ的消费者
消息接收者

##### rabbitmq-producer：RabbitMQ的生产着
消息发布者
